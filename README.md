# Express-REST-middleware-for-Currency-converter-web-application

_Express REST API server application for currency conversion application._

## Requirements

1. node
2. npm
3. typescript

## Installation

```sh
npm install
```

## Build

```sh
npm run build
```

## Start server

```sh
npm start
```

## Development server

```sh
npm run dev
```

## Generate soap client stub

```sh
npm run stub-gen
```

React Build ( `html`,`js`,`css` ) also hosted here: [http://localhost:4000/](http://localhost:4000/)

Docs: [http://localhost:4000/docs](http://localhost:4000/docs)
